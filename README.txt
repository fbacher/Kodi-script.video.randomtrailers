This project is a resurection of the Kodi Random Trailers addon which has
been dormant since 2014 and which has not been functional for some time.

The goals of this project are:

1) To make the addon functional once more

2) To upgrade to Python 3 with Python 2 compatability

3) To modernize the code base: comply with Kodi and Python coding standard
as well as best practices.

4) To improve functionality and performance.
